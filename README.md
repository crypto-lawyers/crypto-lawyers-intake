# The Crypto Lawyers - Webull Case WB156-26

## Project Files

### Website
- `index.html` - Main intake form (6 sections)
- `thank-you.html` - Post-submission confirmation
- `retainer.html` - Signable legal retainer agreement
- `clearing.html` - Financial clearing form with tiered pricing

### Email Campaign
- `bulk_email.py` - Bulk email sender script
- `CUSTOMERS_EMAILS_REMAINING.csv` - 2,118 remaining contacts
- `sent_emails.txt` - 226 sent emails
- `failed_emails.txt` - Failed email attempts

### Assets
- `logo.webp` - Firm logo
- `favicon.webp` - Firm favicon

### Setup
- `apps-script.js` - Google Apps Script code
- `SETUP.txt` - Google Sheets setup guide

## Quick Start

1. **Website:** Open `index.html` in browser
2. **Email:** Run `python bulk_email.py`
3. **Forms:** Follow `SETUP.txt` for Google Sheets backend
