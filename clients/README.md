# CLIENTS FOLDER

Each client who responds to the email campaign gets their own folder.

## Folder Structure

```
clients/
├── README.md
├── matthew-arbogast/
│   └── intake.md
├── [next-client]/
│   └── intake.md
└── ...
```

## Naming Convention
- Folder name: `firstname-lastname` (lowercase, hyphenated)
- Each folder contains `intake.md` with client details

## Client Intake Checklist
1. Create folder with client name
2. Add intake.md with all known details
3. Track evidence received
4. Track retainer status
5. Track clearing form status
